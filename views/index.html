API Project: Timestamp Microservice

Programmer: Brendan Musick

Type in the following after the url name to get a JSON returned date printout, GET [url]/api/timestamp/:date_string? with the date string in one of two formats:
1) 2015-12-25
2) 1479663089000

If the date_string is left empty, it will return the JSON form of todays date.
